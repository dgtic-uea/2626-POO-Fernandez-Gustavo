```
1. Registrar producto       ← Ingresa datos y crea objeto

3. Buscar producto          ← Busca por criterio
```

## Estructura de Carpetas

```
restaurante_app/
├── modelos/
│   ├── __init__.py
│   ├── producto.py      (Clase Producto con @property/@setter)
│   └── cliente.py       (Clase Cliente con @dataclass)
├── servicios/
│   ├── __init__.py
│   └── restaurante.py   (Clase de servicio)
└── main.py              (Menú interactivo)
```

---

## Conceptos Clave Demostrados

| Concepto | Archivo | Ejemplo |
|----------|---------|---------|
| Constructor tradicional | producto.py | `def __init__(self, nombre, ...)` |
| @property | producto.py | `@property def nombre(self)` |
| @setter | producto.py | `@nombre.setter def nombre(self, valor)` |
| @dataclass | cliente.py | `@dataclass class Cliente` |
| Validación | producto.py | `if not valor: raise ValueError` |
| Encapsulación | restaurante.py | `self._productos = []` |
| Búsqueda | restaurante.py | `[p for p in self._productos if...]` |
